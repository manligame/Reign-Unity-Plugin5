Step 1
Declare a "Ad" object to keep track of and control your Ad states. 
private static Ad ad;
Step 2
In your Ad init method declare a "AdDesc" object that will be used to init your "Ad" object. 
var desc = new AdDesc();
Step 3
Now set the basic settings needed for testing your Ad. 
desc.Testing = true;
desc.Visible = true;
Step 4 (Editor)
We can test Ads in the Editor (Look at the AdsDemo scene and code for more info).
Set these values for this target platform if needed. 
desc.Editor_AdAPI = AdAPIs.EditorTestAd;// The target API for your Ad (EditorTestAd and MillennialMediaAdvertising supported)
desc.Editor_AdGravity = AdGravity.BottomCenter;// The position of your Ad
desc.Editor_AdScale = 1.5f;// The scale of your Ad

// If we are targeting MM-Ads set these.
desc.Editor_MillennialMediaAdvertising_APID = "";
desc.Editor_MillennialMediaAdvertising_AdGravity = AdGravity.BottomCenter;
Step 4 (WinRT)
Set these values for this target platform if needed.

desc.WinRT_AdDuplex_AdSize = WinRT_AdDuplex_AdSize.Wide_728x90;

Step 5
Now create the Banner Ad and wait for the callback to fire before using it! 
ad = AdManager.CreateAd(desc, adCreatedCallback);
